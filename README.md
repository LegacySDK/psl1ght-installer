# psl1ght-installer
Installer for PSL1GHT, spu-gcc, ps3-elf-gcc and more!

### Run setup.sh in shell/ if you want to setup everything quickly, use the C version if you absolutely want it to work
Follow the instructions in c/README.md to setup the C version

install via sh script:
```sh
curl -fsSL https://miskaa.pl/legacysdk/sh/setup.sh | sudo bash
```
